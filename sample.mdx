# Markdown with GFM and MDX v1

이 Markdown 문서는 **GitHub Flavored Markdown (GFM)**과 **MDX v1**의 기능을 함께 사용하는 예제입니다.

## GitHub Flavored Markdown (GFM) 기능

### 작업 목록

- [x] 작업 1
- [ ] 작업 2
- [ ] 작업 3

### 사용자 언급

GitHub 사용자를 언급할 수 있습니다: @사용자명.

### 이모지 :smile: :rocket:

GFM에서 이모지를 사용할 수 있습니다! :+1:

## MDX v1 기능

MDX를 사용하면 Markdown 내에서 JSX(React) 구성 요소를 포함할 수 있습니다.

### 내장된 React 구성 요소

다음은 내장된 React 구성 요소입니다:

<CustomComponent prop1="값1" prop2="값2" />
